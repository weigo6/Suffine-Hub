# 友链申请指南

欢迎访问我的博客！👋

如果你也拥有一个技术、学习或生活类的独立博客，并希望与本站交换友情链接，请阅读以下申请指南。

## :material-checkbox-marked-circle-auto-outline: 申请条件

为了保证友链质量与内容相关性，申请友链需满足以下基本条件：

!!! note "申请要求"

    - **博客内容原创**：内容以原创为主，谢绝采集站、营销号、纯转载站点。
    - **持续更新**：近半年内有更新记录，表明你仍在积极维护博客。
    - **独立域名**：不接受免费二级域名（如 `xxx.github.io`、`xxx.netlify.app`除外，但需有自定义域名更佳）。
    - **界面整洁可用**：页面排版合理、无过多广告或恶意跳转。

如果您想申请交换友链，请在留言中包含站点名称、站点地址、站点描述、图标链接（Avatar/Logo）。

## :material-pencil-plus-outline: 申请方式
请通过以下任一方式提交你的友链申请：

### 通过博客表单填写申请 :material-creation-outline:
通过导航栏或者[点击此处](../basic/contact-form/)进入联系表单填写界面，您的个人信息不会被公开。

### 在本指南的评论区留言
本站提供了两套评论系统，你可以通过切换模式按钮选择使用Github登录评论还是匿名评论。
> 匿名评论方式出于隐私保护考虑，请不要伪造他人的评论留言。

### 在 GitHub 提 Issue
如果你熟悉 GitHub，欢迎在本博客的[源码仓库](https://github.com/weigo6/Suffine-Hub)提交一个新 Issue，标题内容为友链申请。

## :material-vector-link: 本站信息
如果你希望将本站加入你的友链列表，可参考以下信息：

    ```yaml title="友链格式示例/本站信息"
    站点名称: Suffine Hub
    站点地址 (URL): https://sufine.top
    站点描述: 分享学习路程，表态生活。
    ```

## :material-timer-sand-complete: 处理时间
我会在 3 个工作日内审核你的申请。若符合条件，我将：

在本站友链页面添加你的站点，并通过邮件或 Issue 回复确认。

## :material-email-open-heart-outline: 温馨提醒
友链是双向的，请在申请前确保你已将本站加入你的友链页面，或承诺在互换后添加。若你的博客长期处于无法访问状态或内容方向涉及违法内容，我会将友链移除，敬请理解。

感谢你对独立博客生态的支持！期待与你链接，共赴数字世界的星辰大海 🌌
