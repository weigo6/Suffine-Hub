# Typora 主题记录

## Typora 原生主题

Typora 官方默认主题仓库：[typora-default-themes](https://github.com/typora/typora-default-themes)

### Whitey

- **风格**：经典、简洁、高对比度、白底主题
- **适用场景**：日常写作、文档编辑、长时间打字
- **特点**：纯白色背景搭配黑色文字，界面清爽干净，符合传统文本编辑器的视觉习惯。适合在光线充足的环境下使用，对眼睛负担较小。

### Pixyll

- **风格**：极简主义、现代印刷风
- **适用场景**：技术写作、博客草稿、长文创作、知识笔记
- **特点**：浅灰白色背景搭配深灰色正文，减少视觉疲劳。精心调校的字体、行高与段落间距，提升可读性。无边框、无阴影、无多余装饰，极致简约。

### Night

- **风格**：深色、护眼、低亮度
- **适用场景**：夜间写作、暗光环境、减少蓝光刺激
- **特点**：深灰色背景搭配浅灰色或白色文字，有效降低屏幕亮度，减少视觉疲劳。代码块、引用、列表等元素均采用柔和的对比色，确保可读性。

### Newsprint

- **风格**：报纸排版、专业、正式
- **适用场景**：新闻稿、报告撰写、学术写作
- **特点**：模拟传统报纸的版式设计，行距适中，字体偏正式，强调段落结构和可读性，适合需要严谨格式的文本类型。

### Github

- **风格**：技术感、极简、开发者友好
- **适用场景**：技术文档、代码笔记、开源项目写作
- **特点**：模仿 GitHub 网站的 Markdown 渲染风格，代码高亮配色与 GitHub 一致，表格、列表等元素排版紧凑，深受程序员喜爱。

## Typora 第三方主题

关于主题的相关设置：[About Themes - Typora Support](https://support.typora.io/About-Themes/)

### Typora-latex-theme

简化中国大陆本科生**小型通识课论文**（或**小型实验报告**）撰写的负担，基本采用了浙江大学要求的格式（字体较小，页边距较小）。将 Typora 伪装成 LaTeX 的中文样式主题，本科生轻量级课程论文撰写的好帮手。

主题项目地址：**[typora-latex-theme](https://github.com/Keldos-Li/typora-latex-theme)**

安装的版本：`v0.3.2`

包含以下两个主题文件：`latex.css` 、`latex-dark.css`

### Everforest

Typora 的一个干净、自然灵感的主题，基于**EverForest**调色板。旨在轻松进行长时间的写作，具有平衡的对比度和和谐的色彩。

主题介绍：[Everforest](https://theme.typora.io/theme/Everforest/)

主题项目地址：**[everforest-typora](https://github.com/EthanBao27/everforest-typora)**

安装的版本：`v1.1.0`

包含以下两个主题文件：`everforest-light.css` 、`everforest-dark.css`

### Blackout Series Theme

**Blackout Series Theme**包含Blackout Theme、Chernobyl Theme两款系列主题。

界面美观，且适用于一体化模式，拥有着清晰的文字和优雅的圆角，十分成熟的**夜间主题**。

Blackout 主题为避免在写作时图片占据大量空间，插入的图片会被缩放至合适大小。为避免写作位置过于狭窄，页面滚动完后仍然能往下滚一段距离。

Blackout Gamer 主题是一个适用于展示的主题，阅读体验很舒适且充满现代感。这个主题不会缩小图片，但是继承了 Blackout 主题优美的圆角设计。

Chernobyl Theme 是一款优雅的现代主题，基于主题 Blackout Gamer。此主题大量使用了毛玻璃效果，具有独特的神秘感和科幻氛围。该主题拥有绿色、蓝色两种配色选择。

主题介绍：[Blackout](https://theme.typora.io/theme/Blackout/)、[Chernobyl](https://theme.typora.io/theme/Chernobyl/)、[黑障主题](https://obscurefreeman.github.io/typora_theme_blackout/zh/)、[切尔诺贝利主题](https://obscurefreeman.github.io/typora_theme_blackout/zh/chernobyl)

主题项目地址：**[typora_theme_blackout](https://github.com/obscurefreeman/typora_theme_blackout)**

安装的版本：`v1.0.9`

包含以下四个主题文件：`blackout.css` 、`blackout-gamer.css`、`chernobyl.css`、`chernobyl-blue.css` 及文件夹`obscurefreeman`

### Ceylon

**Typora Ceylon Light Theme**是为Typora设计的一款**干净、轻便**的主题。灵感高度来源于令人惊叹的Vue轻量级主题和Obsidian编辑器默认主题。

主题介绍：[Ceylon](https://theme.typora.io/theme/Ceylon/)

主题项目地址：**[typora-ceylon-theme](https://github.com/tdarshana/typora-ceylon-theme)**

安装的版本：`v1.2.0`

包含以下三个主题文件：`ceylon.css` 、`ceylon-dark.css`、`ceylon-darker.css` 及文件夹 `ceylon`

### SprIng

一款令人**如沐春风**的 Typora 灵动主题，在设计上使用充满春天气息的配色卡，具有不同效果的灵动交互和风格统一的 Typora 界面。

主题介绍：[SprIng](https://theme.typora.io/theme/SprIng/)

主题项目地址：**[typora-spring-theme](https://github.com/SprInec/typora-spring-theme)**

安装的版本：`v1.1.1`

包含以下一个主题文件：`spring.css`
