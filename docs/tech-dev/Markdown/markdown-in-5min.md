# 五分钟掌握 Markdown 语法

Markdown 是一种轻量级的标记语言，语法简洁、易读易写，广泛用于文档编写、博客撰写、代码注释以及各类支持富文本的平台（如 GitHub、Typora、Obsidian 等）。本文将带你用五分钟快速掌握 Markdown 的核心语法。

---

## 1. 标题（Headers）

使用 `#` 表示标题，数量代表层级（1~6 级）：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> 💡 通常一篇文档以一个 `#` 开头的一级标题作为主标题，后续用二级、三级标题组织内容结构。

---

## 2. 文本样式（Text Formatting）

- **加粗**：用两个星号包裹文字  
  ```markdown
  **加粗文字**
  ```

- *斜体*：用一个星号或下划线  
  ```markdown
  *斜体文字* 或 _斜体文字_
  ```

- ***加粗 + 斜体***：用三个星号  
  ```markdown
  ***加粗且斜体***
  ```

- ~~删除线~~：用两个波浪线  
  ```markdown
  ~~删除线文字~~
  ```

- `行内代码`：用反引号包裹  
  ```markdown
  使用 `console.log()` 输出信息
  ```

---

## 3. 链接与图片（Links and Images）

### 链接
```markdown
[百度](https://www.baidu.com)
[带提示的链接](https://github.com "访问 GitHub")
```

### 图片
```markdown
![替代文字](/path/to/image.jpg)
![带标题的图片](/images/logo.png "公司 Logo")
```

> 📌 注意：图片语法与链接几乎相同，只是前面多一个感叹号 `!`。

---

## 4. 列表（Lists）

### 无序列表
```markdown
- 项目一
- 项目二
  - 子项目
  - 另一个子项目
```

### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

> 🔢 有序列表的数字实际可全写成 `1.`，渲染后仍会自动编号。

---

## 5. 引用（Blockquotes）

用于引用他人话语、重点提示或注意事项：

```markdown
> 这是一段引用文字。
> 可以跨多行。
>> 还支持嵌套引用。
```

---

## 6. 代码块（Code Blocks）

用三个反引号包裹代码，并可指定语言以支持语法高亮：

````markdown
```python
def greet(name):
    print(f"Hello, {name}!")
```

```javascript
console.log("Hello from JS!");
```
````

---

## 7. 表格（Tables）

用竖线 `|` 分隔列，第二行用短横线 `---` 定义对齐方式：

```markdown
| 姓名   | 年龄 | 城市     |
|--------|------|----------|
| 张三   | 28   | 北京     |
| 李四   | 22   | 上海     |
```

> 📏 默认左对齐；如需居中或右对齐，可在分隔线中加入冒号：
> - `:---` 左对齐（默认）
> - `:---:` 居中
> - `---:` 右对齐

---

## 8. 分割线（Horizontal Rule）

三种写法均可生成水平线：

```markdown
---
***
___
```

常用于章节分隔。

---

## 9. 任务列表（Task Lists）

适用于记录待办事项（在 GitHub、Notion 等平台支持）：

```markdown
- [x] 完成 Markdown 学习
- [ ] 撰写技术博客
- [ ] 优化个人简历
```

---

## 10. 转义字符（Escaping）

当需要显示 Markdown 语法符号本身（而非其功能）时，用反斜杠 `\` 转义：

```markdown
显示星号：\*这不是斜体\*
显示井号：\# 这不是标题
```

常见需转义的字符：`\ ` `*` `_` `{}` `[]` `()` `#` `+` `-` `.` `!` `` ` ``

---

## 11. 换行与段落（Line Breaks & Paragraphs）

- **软换行**：在行尾加两个或以上空格，再回车  
  ```markdown
  第一行（末尾有两个空格）  
  第二行（与上一行在同一段落内）
  ```

- **新段落**：用空行分隔  
  ```markdown
  这是第一段。

  这是第二段。
  ```

---

## 小结

Markdown 以“所见即所得”的理念，让你专注于内容本身，而非排版细节。掌握以上语法，即可高效撰写结构清晰、格式美观的技术文档、博客文章或笔记。

> ✅ **建议**：在支持实时预览的编辑器（如 Typora、VS Code + Markdown 插件、Obsidian）中边写边看效果，学习效率更高！
