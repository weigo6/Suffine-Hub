# 卡西欧 fx-991 计算器使用技巧

在日常学习、工程计算乃至研究生入学考试（以前可以使用计算器，现在绝大多数的院校都应该用不了了，不过在备考过程中还是会很有帮助）中，科学计算器是不可或缺的工具。

本文将梳理了科学计算器的**按键颜色规则**，并重点介绍几个**高频实用技巧**。

## 一、按键颜色规则详解

科学计算器的物理按键表面通常标注**三种颜色**的符号，分别对应不同的输入层级。掌握这一规则，是高效使用计算器的前提。

- **白色字符**：直接按下即可输入。这是最基础的操作方式，适用于数字、常规运算符（如 `+`, `-`, `×`, `÷`）、等号 `=` 等。
  
- **黄色字符**：需**先按左上角的 `SHIFT` 键**（部分机型标注为 `2ndF` 或 `Inv`），此时屏幕左上角会显示 **`S`** 标志，表明已进入“第二功能”模式。随后再按对应按键，即可调用其黄色标注的功能（如 `sin⁻¹`, `√`, `10ˣ` 等）。

- **红色字符**：需**先按左上角的 `ALPHA` 键**，此时屏幕左上角会显示 **`A`** 标志，进入“字母/变量”模式。此模式常用于输入变量（如 `X`, `Y`）、存储数据、或调用特定内存函数。

!!! note ""
    白色字体的可以直接按出来显示。

    黄色字体的功能通过按左上角的“SHIFT”键来使用，按下此键显示屏上会出现一个S标志，此时可使用黄色功能。

    红色字体的功能通过按左上角的“ALPHA”键来使用，按下此键显示屏上会出现一个A标志，此时可使用红色功能。

> 💡 **小提示**：`SHIFT` 与 `ALPHA` 均为“临时切换键”，按下后仅对**下一次按键**生效，操作完成后自动退出。若需连续使用多个黄色或红色功能，需重复按 `SHIFT`/`ALPHA`。

## 二、高频实用技巧分享

### 1. 复数形式转换：从直角坐标 `a + bi` 到极坐标 `r∠θ`

在电路分析、信号处理或复变函数等课程中，常需在复数的不同表示形式间切换。以主流型号（如 Casio fx-991CNX / fx-991ES PLUS）为例：

#### 操作步骤

1. **进入复数模式**  
    按下 `菜单（MENU）` → 选择 `2`（复数计算模式）。此时屏幕顶部会出现 **`i`** 标志，表示已启用复数功能。

2. **设置输出格式为极坐标**  
    - 按 `SHIFT` + `菜单（SETUP）` 进入设置界面；
    - 按 `↓`（方向键下翻）至“复数”选项（通常为第2项）；
    - 选择 `2`（即 `r∠θ` 形式）作为输出格式。

3. **输入并转换**  
    直接输入复数表达式，如 `3 + 4i`，然后按 `=`，计算器将自动显示其模长 `r` 与幅角 `θ`（默认单位为当前角度模式）。

> ⚠️ **注意**：若未提前设置输出格式，结果可能仍以 `a + bi` 形式显示，需手动切换。

!!! note "将一个复数a+bi——>r∠θ"
    1) 按下“菜单”，再按数字2，按下后显示屏上会出现一个i标志

    2) 按下“SHIFT”再按“设置“（菜单键的位置），切换到设置，按”方向键下翻“，按2选择第二项“复数”，选择输出的形式“1或2”，此时按下2

    3) 设置完成后输入a+bi，按下“=”

### 2. 分数与小数快速互转：`S⇔D` 键的妙用

在概率统计、极限计算或数值近似中，常需在精确分数与十进制小数间切换。

#### 使用方法

- 先输入表达式（如 `1 ÷ 3`）并按 `=`，得到结果（如 `1/3` 或 `0.333333333`）；
- 直接按 **`S⇔D`** 键（通常位于屏幕下方或中部），即可在分数与小数之间**一键切换**。

> ✅ **优势**：避免重新输入，且保留原始精度。例如，`√2` 的精确值不会因转换而损失。

### 3. 角度与弧度的智能转换

有些理工科考试要求答案以**弧度**形式给出，而题目可能以角度给出。此时需灵活切换单位。

#### 操作流程

1. **确认当前角度模式**  
    - 屏幕顶部若显示 **`D`**，表示当前为**角度制**（Degree）；
    - 若显示 **`R`**，则为**弧度制**（Radian）；
    - 可通过 `SHIFT` + `菜单（SETUP）` → `2：角度单位` → 选择 `1（Deg）` 或 `2（Rad）` 进行切换。

2. **显式标注单位进行转换**  
    - 假设当前在 **角度模式（D）**，想将 `60°` 转为弧度：
      - 输入 `60`；
      - 按 `SHIFT` + `OPTN`（或 `MODE`，依机型而定）；
      - 选择 `2：角度单位` → 选择 `1：°`（角度符号）；
      - 此时屏幕显示 `60°`；
      - 再按 `SHIFT` + `OPTN` → 选择 `2` → `2：r`（弧度符号），屏幕变为 `60°r`；
      - 按 `=`，即可得到弧度值 `π/3 ≈ 1.0472`。

    - **反向操作**（弧度→角度）同理：在 `R` 模式下输入 `π/2`，附加 `r` 单位，再转为 `°` 单位后按 `=`。

!!! note "角度弧度转化"

    > 1) 首先要在角度模式显示字母D时输入待转换的值（如果是弧度转角度则需要在显示R时输入，这个可以通过shift+菜单/设置进入“2：角度单位”来切换）

    > 2) 按下shift下方的OPTN按键，按2选择第二项“角度单位”，选择单位形式“角度°，弧度r或百分比g”，此时按下2，显示屏上输入的值上会出现一个r标志，具体显示内容如图

    > 3) 此时按下"="即可转换结果，弧度转角度同理可以操作。
